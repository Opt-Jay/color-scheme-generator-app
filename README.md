## 🚀 Overview

The **Color Scheme Generator** is a modern web application that allows users to generate harmonious color palettes from a selected base color.

It integrates with an external API to dynamically fetch color schemes and provides an intuitive UI for quick selection and copying.

---

## ✨ Features

- 🎯 Select a base color using a native color picker  
- 🎨 Generate multiple color schemes (monochrome, triad, complement, etc.)  
- ⚡ Fast API-powered palette generation  
- 📋 Click any color to copy HEX values instantly  
- 📱 Fully responsive (mobile-first design)
